# Insurance Cost Prediction

A Machine Learning project that predicts medical insurance costs using demographic and health-related features.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Feature Scaling
- Model Training
- Model Evaluation

## Results

The model was trained using Linear Regression and evaluated using R² Score and MAE.
